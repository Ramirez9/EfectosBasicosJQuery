Guía sobre los efectos con sus velocidades y complete

Parámetros comúnes de los efectos

duration (opcional) --> Velocidad de la animación en milisegundos sus valores son "slow(600)", "fast(400)" o "(número)"

easing (opcional) --> Tipo de animación "linea" o "swing (defecto)" linear sería una animación constante y swing con cambios.

callback (opcional) --> Función a ejecutarse una vez finalizada la animación

Efecto .hide() y .show()

Se encargan de ocultar y mostrar respectivamente elementos del DOM.

El uso de estas dos emulan la funcionalidad básica de .toggle()

Su sintaxis es la siguiente: $(selector).hide(duration [,easing][,callback]

Efecto .toggle

Se usa en casos para ocultar y mostrar elementos de una página web

Su sintaxis es la siguiente: $(selector).toggle(duration [,easing][,callback]

También dispone de un display: .toggle (display), false = hide, true = show
